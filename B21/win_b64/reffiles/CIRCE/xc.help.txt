Knowledge Base Editor
---------------------
  
Here are the options available from the menu bar:
FILE file management
    SAVE          Save the knowledge base content in the backup file
    SAVE AS       Save the knowledge base content in any file
    RESTORE       Restore the knowledge base initlai content
    CLOSE         Iconifies the knowledge base browser to the main window
  
TEST
    RUN           Runs the knowledge base
    LOAD FACTS    Loads facts from an express file
    SAVE FACTS    Saves facts to an express file
    DELETE FACTS  Deletes all the facts from the knowledge base
    TRACE OPTIONS
       CLASSES       Displays the panel for tracing classes
       VARIABLES     Displays the panel for tracing variables
       RULES         Displays the panel for tracing rules
       ALL CLASSES   Give a common trace level for all classes
       ALL VARIABLES Give a common trace level for all variable
       ALL RULES     Give a common trace level for all rules
       AGENDA        Agenda management (*)
  
CONFIG
    COMPILER OPTIONS Sets the circe compiler options (*)
    EXTERNAL OBJECTS Use external object files for application link (*)
  
HELP
    HELP          Displays this help message
  
  
Edit the knowledge base name text field
  
Press the rightmost mouse button to display popup menus.
  
  
  
(*) Not Available
