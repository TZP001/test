CATIA V5 Print Driver Plugin Interface

 

Quick Implementation Description:

The 10 following functions must be overwritted:
  • CATPDPError CATPDP_Begin(const CATPDPParameters& param)
  • CATPDPError CATPDP_End(void)
  • CATPDPError CATPDP_DefineColor(int iIndex, float iR, float iG, float iB)
  • CATPDPError CATPDP_SelectDrawColor(int index)
  • CATPDPError CATPDP_SetDrawWidth(float width)
  • CATPDPError CATPDP_MoveTo(float x, float y)
  • CATPDPError CATPDP_LineTo(float x, float y)
  • CATPDPError CATPDP_SelectFillColor(int index)
  • CATPDPError CATPDP_FillArea(int, const int*, const float*)
  • CATPDPError CATPDP_DrawBitmap(float x, float y, const int type, const unsigned char * pixels, const int size)

To help you constructs your library, a template file and Samples of driver implementation are delivered.
CATIA_install\startup\PrintServices\PrintDriverPlugIn

Depends of your OS implementation, use makefile template to help you build your library module.

Step-by-Step description:
  • Add your code for each of the 10 functions
  • Build a DLL or .so module (refers to the Makefile delivered as sample)
  • After construction, declares the resulting module (DLL or .so) to the V5 Printer Driver Manager File / Printer.
    • Setup or Start->Programs->CATIA->Tools->Printer Setup
    • Add a 3DPLM Printer and customize as a V5 Printer Driver Plugin
    • As parameter specifies the library name and path of the Printer Plugin Driver Module (DLL or .so)
