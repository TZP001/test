History

13-01-2006: (0602) Small change to the NC_TOOL_CHANGE syntax to get constant
    output for PPRINT with all type of NC_TOOL_CHANGE

11-11-2005: (0545) Fixed the NC_HEAD_ROTATION and NC_TABLE_ROTATION
    that used an invalid ROTABL syntax (removed ATANGL). Add back
    NC_COUNTERBORING that was lost when mill and lathe pptables were
    merged. NC_BORING_SPINDLE_STOP and NC_BORING_AND_CHAMFERING also
    got their definition mixed up. Fixed them. NC_BORING_AND_CHAMFERING
    is not supported at the moment and will generate a warning. Also
    changed all use of PPRINT and PARTNO. These command do not issue
    the '/' anymore to match CATIA PPRINT and PARTNO syntax that are
    output without the use of the pptable.

07-10-2005: (0540) Merged the mill and lathe pptables. Changed the 
    NC_ORIGIN syntax to use CUTCOM insted of ORIGIN. Added major word
    LCS and minor word WIND. Reviewed NC_START_MACRO. The NC_TOOL_CHANGE
    now also output COOLNT statement as defined in tool parameter pannel.
    A new dummy macro was created in the CAM-POST/CATIA interface kit to
    catch any empty COOLNT statement and prevent warnings. The
    NC_TOOL_CHANGE_LATHE default syntax now output the OSETNO with the
    TURRET command.

22-07-2005: (0529) Added NC_AXIS_LOCK and NC_AXIS_MOVE.

19-04-2005: Added support for circular milling and thread milling. Also
    changed the cycle syntax to support a jump distance. The circular
    milling and thread milling commands require the use of a CATIA
    interface kit macro "kit.obj".

14-04-2005: LOADTL and TURRET syntax sequences 1 and 2 have been swapped
    so the default tool change syntax no longer includes the OSETNO
    parameter. This was done because the MFG_TOOL_COMP value is not always
    correctly defined by CATIA at a tool change.

19-01-2004: Updated to better support mills, lathes and mill/turns. The
    CYCLE syntax has also been modified so that the DWELL cycle option can
    optionally appear on supported cycles.

05-07-2002: First version of the PPTable for CATIA V5R9SP2.
