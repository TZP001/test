ge_7544_lathe4 demonstration post-processor

The ge_7544_lathe4 post-processor produces output for a generic 4-axis dual merging turret lathe with a General Electric GE-7544 controller.

The cross-slide control is output as a radius value.

Motions and commands following a HEAD/MAIN will be output to the ZX axes. Those following a HEAD/SIDE will be output to the WU axes. The HEAD command can also be used to output simultaneous 4-axis merged motions (follow the post-processor documentation link below for a brief description of the HEAD and other post-processor commands).

This post-processor outputs INCH units only.

This is a demonstration post-processor - output is limited to 300 blocks.
