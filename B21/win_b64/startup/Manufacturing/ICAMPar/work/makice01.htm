MAKICE01

The MAKICE01 CE can be use to simulate ISO NC code for this Makino machine with a Fanuc 16M controller. To run the example first load the sample CATProcess. Using the right mouse button on the manufacturing program in the PPR tree (0001) select from the popup context menu 'Simulate Machine using NC code'.

Note: This sample can only be run on Windows 32-bit workstations.


Machine Type: Milling - 5 Axis continuous - X Y Z with A and C dual rotary tables
Machine Details: Makino 5x
Controller Details: Fanuc 16M
Operation Units: MM
Sample CatProcess: smile.CATProcess
