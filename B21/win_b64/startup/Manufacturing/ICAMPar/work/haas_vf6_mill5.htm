haas_vf6_mill5 demonstration post-processor

The haas_vf6_mill5 post-processor produces output for a generic 5-axis mill with a HAAS Automation VF-6 controller.

This generic machine has 2 rotary tables. The lowest table (ie. furthest from the part) rotates about the y-axis and is controlled by the B register. The highest table (ie. nearest to the part) rotates about the z-axis and is controlled by the C register.

This post-processor outputs INCH units only.

This is a demonstration post-processor - output is limited to 300 blocks.
