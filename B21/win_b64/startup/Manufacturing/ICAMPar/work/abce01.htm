ABCE01

The ABCE01 CE can be used to simulate ISO NC code for this 5-axis machine with an Allen Bradley 8400 controller. To run the example, first load the sample CATProcess. Then, using the right mouse button on the manufacturing program in the PPR tree (Manufacturing Program.1), select 'Simulate Machine using NC code' from the popup context menu.

This controler can use up to 99 different tools and tool compensations and up to 12 different fixture compensations.

There are two sample process available for this CE demo. The first one called SampleProcess demonstrates a 5 axis machining process. The second process called FixtureComp demonstrates the use of fixture compensation via G45 O1 to G45 O4.

Note: This sample can only be run on Windows 32-bit workstations.


Machine Type: Milling - 5 Axis continuous - X Y Z with A and B Dual Rotary Head
Machine Details: Generic 5x
Controller Details: ALLEN BRADLEY 8400
Operation Units: MM
Sample CatProcess: SampleProcess.CATProcess
FixtureComp CatProcess: FixtureComp.CATProcess
