mazak_tplus_lathe3 demonstration post-processor

The mazak_tp_lathe3 post-processor produces output for a generic 3-axis (z,x,c) lathe with a Yamazaki Mazak Corporation Mazatrol T-Plus controller.

The cross-slide control is output as a diameter value.

Motions and other post processor commands following an APPLY/MILL will be output as ZXC values using the live spindle. Those following an APPLY/TURN will be output as ZX values using fixed tooling.

This post-processor outputs MM units only.

This is a demonstration post-processor - output is limited to 300 blocks.
