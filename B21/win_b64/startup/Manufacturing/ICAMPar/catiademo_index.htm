Use the None post-processor selection (in the NC Code tab page) to activate the CAM-POST launch panel. This panel (shown below) can be used to select alternate CL files, post-processors (and post-processor databases), as well as to select different program options and run-time display modes. This same panel will appear when the MoreOptions checkbox is selected (also in the NC Code tab page).

3-Axis Milling

The iso_6983_mill3 post-processor produces output for any 3-axis mill controller that adheres to International Standard ISO 6983-1988 "Numerical control of machines - Program format and definition of address words". This standard defines (G) and (M) functions for various classes of manufacturing machines. This post-processor outputs either MM or INCH units, depending on the input APT file units. This is a demonstration post-processor with unlimited output.

5-Axis Milling

The cin_ecs2401d_mill5 post-processor produces output for a generic 5-axis mill with a Milacron ECS 2401D controller. This generic machine has 2 rotary heads. The highest head (ie. furthest from the tool) rotates about the y-axis and is controlled by the B register. The lowest head (ie. nearest to the tool) rotates about the x-axis and is controlled by the A register. This post-processor outputs MM units only. This is a demonstration post-processor - output is limited to 300 blocks.

3-Axis Mill/Turn Lathe

The mazak_tp_lathe3 post-processor produces output for a generic 3-axis (z,x,c) lathe with a Yamazaki Mazak Corporation Mazatrol T-Plus controller. The cross-slide control is output as a diameter value. Motions and other post processor commands following an APPLY/MILL will be output as ZXC values using the live spindle. Those following an APPLY/TURN will be output as ZX values using fixed tooling. This post-processor outputs MM units only. This is a demonstration post-processor - output is limited to 300 blocks.

4-Axis Merging Lathe

The ge_7544_lathe4 post-processor produces output for a generic 4-axis dual merging turret lathe with a General Electric GE-7544 controller. The cross-slide control is output as a radius value. Motions and commands following a HEAD/MAIN will be output to the ZX axes. Those following a HEAD/SIDE will be output to the WU axes. The HEAD command can also be used to output simultaneous 4-axis merged motions (follow the post-processor documentation link below for a brief description of the HEAD and other post-processor commands). This post-processor outputs INCH units only. This is a demonstration post-processor - output is limited to 300 blocks.
