Special features
Cutter compensation
---------------------------------
X and Y registers, though modal are always output in a block
which contains G41/G42 functions of a cutter compensation
Register D signifies a radius compensation when output with X and Y
while being output with Z it signifies a tool length correction
Drilling cycles
---------------------
All drilling cycles are moved to the end of NC output and called from within the main program.
Example:
call cycle number 8 at point X10 Y12 Z25 :
N0125 U0008 G00 X+10000 Y+12000 Z+250000 M08
At the end of main program find:
U0008 G81 F.138 Z-28. S+1200 G4 V4.0 Z147.
At the end of each cycle the previous feedrate is being restored
Subroutines
--------------------
N360 L1 N330 N350
L - the current number of a subroutine is followed by block numbers of subroutine start and subroutine end;
