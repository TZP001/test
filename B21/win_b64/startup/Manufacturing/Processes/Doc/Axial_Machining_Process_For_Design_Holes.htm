The Axial Machining Process for Design Holes enables you to apply it on any design hole Feature using different information like hole type, hole diameter, counterbored diameter, tolerances on diameters,  etc..., to execute checks, tooling queries and to valuate NC parameters from design hole or tool parameters.

Detailing this machining process:

  • The spot drilling operation is controlled by the hole diameter which has to be greater than 2 mm. Then, the tool of "Spot Drill D10" name will be extract from the "ToolSampleMP.catalog" tools catalog

  • For the drilling break chips operation, the tool is chosen according the nominal diameter less than the hole diameter. The drilling depth is automatically computed from the hole depth from the default mapping mechanism.

  • The reaming operation is controlled by the "H7"quality defined on the hole and affected to its diameter: if you receive an error message including "Hole.diameter.SemanticDimTabValue", it means that the "SemanticDimTabValue" is not accessible. Then, you have to load the "TPSPackage" package in the Tools+Options menu item and
