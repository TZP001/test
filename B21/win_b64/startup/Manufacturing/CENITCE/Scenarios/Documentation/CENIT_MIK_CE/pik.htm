•NC Resources

Right-click the "ZIMMERMANN_FZ30" node on the PPR tree. Select "NC Resources" and then "Edit NC Resources" from the context menu.

Necessary options set in the "Numerical Control" Tabpage:

# Post & Controller Package: ZIMMERMANN_FZ30_S840D.ce

Additional definitions for ISO Code generation:

# Post Processing: ZIMMERMANN_FZ30_S840D.pp

# Post Processor words table: CPOST_MILL.pptable

Default NC Resources on Machinetool CATProduct, maybe overwritten by individual MFG CATProcess settings.
