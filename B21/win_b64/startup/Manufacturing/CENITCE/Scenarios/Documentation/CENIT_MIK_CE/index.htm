Tool Offset And Clash Detection

This documentation contains the description of a 5 Axis MIK CE Demo scenario with tool offsets and clash examples, provided by Cenit AG Systemhaus.

The scenario can be used in Catia/Delmia ISO Code based simulation environment, to demonstrate the functionality of Cenit MIK CE and to get familiar with the MSG functionality.
