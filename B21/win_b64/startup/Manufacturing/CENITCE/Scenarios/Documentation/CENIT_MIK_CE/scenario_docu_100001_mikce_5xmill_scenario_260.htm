Configure Scenario

This section describes the necessary configuration settings for ISO Code based simulation

•Common Settings
Left-click on Menu Item "Tools" then "Options...". Select "Machining" then "Output" Tabpage.
Set Option "PostProcessor" to "Cenit".

 

MPS Definitions on CATProcess

Load the CATProcess with left-click on Menu "File" then "Open..."

Select the file 100001_PROCESS_5XMILL_SCENARIO.CATProcess from the directory containing the scenario.

 

 

 

Part Operation Definition

Right-click the "100001_PROCESS_5XMILL_SCENARIO" Part Operation node on the PPR tree. Select "100001_PROCESS_5XMILL_SCENARIO.object" and then "Definition..." from the context menu, to open Part Operation Definition Panel:

Select the "Machine" Icon to open the "Machine Editor" panel.

 

 

Machine Editor Panel

 

 

 

Assign a machine from file selection

Select the "Assign a machine from file selection" Icon

Select the ZIMMERMANN_FZ30_S840D.CATProduct:

 

 

NC Resources

Verify / Modify default NC Resource settings:

 

 

Workpiece Automatic Mount

Execute the Command "Workpiece Automatic Mount" from the "Machine Management" Toolbar:

 

Actions done by Delmia on Automount:

a) Attach the Workpiece CATPart/CATProduct to the Machinetool table CATPart/Catproduct

b) Snap/Align the Workpiece CATPart/CATProduct with its NC Axis System to the Workpiece Mount Point of the Machinetool

 

 

Run Scenario

This section describes how to run the scenario.

•Load CATProcess
Open Document "_MPS_ZIMMERMANN_100001_PROCESS_5XMILL_SCENARIO.CATProces

•Generate NC Code Interactively
Right-click the "PO_100001" node on the PPR tree. Select "PO_100001 object" and then "Generate NC Code Interactively"
Execute the command and after successful ISO Code generation, Cancel the "Generate NC Output Interactively" panel.

 

•Set ISO Code Simulation mode
Double Left-click the "PO_100001" node on the PPR tree (Definition...).
Activate the "MCD/ISO Based Simulation" Option

 
ISO Data for simulation is automatically selected:

 
In PPR Tree, the Text on the Program has changed:

•Start ISO Code Based Simulation
Execute the Command "Starts Machine Tool Path Simulation" from the "NC Output Management" Toolbar:

 
Now the Cenit MIK CE is called to generate the simulation data. Delmia changes to the Machine Tool Path Simulation Workbench and opens the "Process Simulation" Toolbar:
Press "Run" to start the simulation.
