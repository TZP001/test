Configure Scenario

This section describes the necessary configuration settings for ISO Code based simulation

•Common Settings
Left-click on Menu Item "Tools" then "Options...". Select "Machining" then "Output" Tabpage.
Set Option "PostProcessor" to "Cenit".
 

 

MPS Definitions on CATProcess

Load the CATProcess with left-click on Menu "File" then "Open..."

Select the file 100001_PROCESS_5XMILL_SCENARIO.CATProcess from the directory containing the scenario.

 

 

Part Operation Definition

Right-click the "100001_PROCESS_5XMILL_SCENARIO" Part Operation node on the PPR tree. Select "100001_PROCESS_5XMILL_SCENARIO.object" and then "Definition..." from the context menu, to open Part Operation Definition Panel:

Select the "Machine" Icon to open the "Machine Editor" panel.

Workpiece Automatic Mount

Execute the Command "Workpiece Automatic Mount" from the "Machine Management" Toolbar:

 

Actions done by Delmia on Automount:

a) Attach the Workpiece CATPart/CATProduct to the Machinetool table CATPart/Catproduct

b) Snap/Align the Workpiece CATPart/CATProduct with its NC Axis System to the Workpiece Mount Point of the Machinetool
