New Workspace

Choose what you want to do first in your new workspace:

  • If you have access to Workspace Manager tools, you can modify an existing framework under Configuration control by attaching it from a reference workspace
  • You can modify an existing framework by directly copying it from a reference workspace
  • You can create a new generic framework to create and store new objects
  • You can create a specific ENOVIA framework to create new ENOVIA objects derived from ENOVIA basic types.

You will be able to run these actions later directly from the menu.
