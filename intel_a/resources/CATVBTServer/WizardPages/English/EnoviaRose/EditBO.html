Edit Business Objects

To create a Business Object (BO):

  1. Select the package where the BO will be created in the left panel
  2. The right panel will display available types (Used package). To change the used package, double click on the package
  3. Double-click on a type in the right panel. A new BO based on this type is created in the left panel.

To add an attribute to a BO:

  1. Select the BO where the attribute will be created in the left panel
  2. The right panel will display available attributes (attributes of the root type on which the BO was created)
  3. Double-click on an attribute in the right panel. A new attribute is added to the selected BO with a one level path.

To extend the path of a BO attribute:

  1. Select the last path item of an attribute, the right panel will display attributes of the selected path item
  2. Double-click on an attribute in the right panel to add the corresponding item to the BO attribute's path.
