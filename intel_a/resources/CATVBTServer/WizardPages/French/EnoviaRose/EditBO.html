Editer des Business Objets

Pour créer un Business Objet (BO) :

  1. Sélectionnez le package où le BO va être créé dans le panel de gauche ;
  2. Le panel de droite va lister les types possibles (Used package). Pour changer de used package, double cliquez sur le package ;
  3. Double-cliquez sur un type dans le panel de droite. Un nouveau BO basé sur ce type est créé dans le panel de gauche.

Pour ajouter un attribut à un BO :

  1. Sélectionnez le BO où l'attribut va être créé dans le panel de gauche ;
  2. Le panel de droite liste les attributs possibles (attribut du type sur lequel le BO a été créé) ;
  3. Double-cliquez sur un attribut dans le panel de droite. Un nouvel attribut est ajouté au BO sélectionné.

Pour déployer le chemin de l'attribut du BO :

  1. Sélectionnez le dernier élément dans le chemin de l'attribut, le panel de droite liste les attributs de l'élément sélectionné ;
  2. Double-cliquez sur un attribut dans le panel de droite pour ajouter l'élément correspondant au chemin de l'attribut du BO.
