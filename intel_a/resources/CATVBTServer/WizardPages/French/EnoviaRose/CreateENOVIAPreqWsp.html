Créer les prérequis ENOVIA

Pour pouvoir utiliser la personnalisation ENOVIA, vous devez accéder à des objets situés sur votre serveur UNIX. Pour transférer ces objets vers Windows :

  1. Indiquez le workspace UNIX ou ENOVIA a été installé ;
  2. Indiquez le nom du workspace NT que vous désirez créer ;
  3. Tapez votre mot de passe UNIX afin de transférer les données de UNIX vers Windows.

Vous ne pouvez utiliser cette commande que si votre connexion UNIX a été établie.
