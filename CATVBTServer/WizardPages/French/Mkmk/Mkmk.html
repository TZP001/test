Constructeur de code (mkmk)

Sélectionner les modules à construire dans l'arborescence.

Options
-u Forcer la mise a jour des dépendances
-g Construire en mode debug (pas d'optimisations)
-w Activer la détection des warnings lors de la compilation
-bsc Générer les fichiers .bsc (navigation avancée dans les sources de Microsoft Visual C++)
-i Continuer en cas d'erreurs
