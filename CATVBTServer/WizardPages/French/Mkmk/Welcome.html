CAA RADE V5R21 - Bienvenue

Addins pour  Visual Studio 2008 SP1
Raccourcis clavier dans Visual Studio:
[F7] Pour construire le module correspondant au source en édition
[Ctrl-Shift-F] Pour récupérer le chemin du source en édition
[Ctrl-Q] Pour passer du .h au .cpp et du NLS francais à anglais ou vice-versa
[Ctrl-T] Pour ouvrir le .h correspondant au mot-clef sous le curseur
[F8] Pour check-out du source en édition (Source Code Manager)
[Ctrl-F1] Pour accéder à la documentation des APIs
[F1] Pour obtenir l'aide contextuel sur les boîtes de dialogue
[Ctrl-Shift-X] Pour lancer l'explorateur de fichiers windows sur le répertoire du source en édition
[Ctrl-PgUp] Pour aller à la méthode précédente dans le source en édition
[Ctrl-PgDown] Pour aller à la méthode suivante dans le source en édition
[Ctrl-Shift-H] Pour inserer l'include du .h associe au symbole sélectionné dans le code
[Ctrl+Shift+UpArrow] Pour aller à l'occurence précédente du symbole sélectionné dans le code
[Ctrl+Shift+DownArrow] Pour aller à l'occurence suivante du symbole sélectionné dans le code
[Ctrl-Shift-Y] Pour avoir des informations supplémentaires de debug sur les objets CATBaseUnknown
