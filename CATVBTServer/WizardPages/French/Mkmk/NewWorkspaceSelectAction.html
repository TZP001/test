Nouvel espace de travail

Sélectionnez ce que vous voulez faire pour commencer dans votre nouvel espace de travail.

  • Si vous avez accès aux outils du Workspace Manager, vous pouvez modifier un framework existant sous le controle du gestionnaire de configuration en le rattachant depuis un espace de travail de référence ;
  • Vous pouvez modifier un framework existant en le copiant directement depuis un espace de travail de référence ;
  • Vous pouvez créer un framework générique pour y créer vos objets ;
  • Vous pouvez créer un framework dédié ENOVIA pour créer des objets ENOVIA dérivés de types de base ENOVIA.

Vous pourrez exécuter ces actions plus tard en y accédant depuis le menu.
