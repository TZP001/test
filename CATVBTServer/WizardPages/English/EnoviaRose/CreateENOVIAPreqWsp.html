Create the ENOVIA Prerequisite Workspace

In order to use the ENOVIA customization, you must access objects that currently reside on your UNIX server. To transfer those objects to Windows:

  1. Select the UNIX workspace where ENOVIA is installed
  2. Specify the name of the Windows workspace that will be created
  3. Enter your UNIX password in order to transfer the data from UNIX to Windows.

You may only use this command if your remote UNIX connection has been activated.
